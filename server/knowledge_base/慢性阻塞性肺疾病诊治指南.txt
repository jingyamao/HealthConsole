慢性阻塞性肺疾病诊治指南（专业医疗知识文档）

一、概述

慢性阻塞性肺疾病（COPD）简称慢阻肺，是一种以持续性气流受限为特征的常见慢性呼吸系统疾病。气流受限呈进行性发展，与气道和肺组织对有害颗粒或气体的异常炎症反应相关。COPD可防可治，其病理学改变主要包括气道（支气管炎、细支气管炎）和/或肺泡异常（肺气肿）。严重的合并症可能影响疾病的表现和病死率。

流行病学数据显示，COPD是全球第三大死因，给患者家庭和社会带来沉重的经济负担。在我国，COPD呈现高患病率、高死亡率和高疾病负担的特征。2018年“中国成人肺部健康研究”结果显示，我国20岁及以上成年人的COPD患病率为8.6%，40岁及以上人群患病率高达13.7%，估算患者人数近1亿。由于人口老龄化、吸烟和二手烟暴露、室内外空气污染等因素持续存在，预计未来患病人数仍将增加，到2030年可能达到约1.10亿。

二、病因和发病机制

COPD的发病是遗传因素与环境因素长期相互作用的结果。

主要危险因素包括：
1. 吸烟：是最重要的环境发病因素，包括二手烟暴露。吸烟者COPD发病率显著高于不吸烟者。
2. 职业性粉尘和化学物质暴露：如矽尘、煤尘、镉等。
3. 空气污染：包括室外空气污染（如PM2.5、二氧化硫）和室内生物燃料燃烧产生的污染。
4. 呼吸道感染：儿童期下呼吸道感染是成年后发生COPD的独立危险因素，呼吸道感染也是导致COPD急性加重的重要诱因。
5. 遗传易感性：α1-抗胰蛋白酶缺乏是明确的遗传危险因素。COPD发病具有多基因遗传特点和家族聚集倾向。
6. 肺发育异常：肺发育过程中的任何影响均可增加COPD风险。

病理生理机制核心是慢性炎症反应。有害颗粒和气体（如香烟烟雾）可激活肺内炎症细胞（如巨噬细胞、中性粒细胞、淋巴细胞），释放多种炎症介质，破坏肺的结构和/或促进炎症反应。此外，氧化与抗氧化失衡、蛋白酶与抗蛋白酶失衡也在发病中起重要作用。这些过程导致小气道病变（气道炎症、纤维化、粘液高分泌）和肺实质破坏（肺泡附着丧失、弹性回缩力下降），最终引起持续性气流受限。

三、临床表现

症状和体征：
COPD多于中年以后发病，好发于秋冬寒冷季节。主要症状包括：
1. 慢性咳嗽：常为首发症状，晨起咳嗽明显，夜间可有阵咳。
2. 咳痰：通常为白色粘液或浆液性泡沫痰，合并感染时痰量增多，转为脓性。
3. 气短或呼吸困难：是COPD的标志性症状，早期仅在劳力时出现，后逐渐加重，以致日常活动甚至休息时也感气短。
4. 喘息和胸闷：部分患者特别是重症患者或急性加重时可出现。
5. 晚期患者可有体重下降、食欲减退、抑郁和/或焦虑等。

体征早期可不明显，随疾病进展可出现：
- 视诊：桶状胸，呼吸变浅、频率增快。
- 触诊：双侧语颤减弱。
- 叩诊：肺部过清音，心浊音界缩小，肺下界下降。
- 听诊：两肺呼吸音减弱，呼气期延长，可闻及干性或湿性啰音。

临床分期：
1. 稳定期：咳嗽、咳痰和气短等症状稳定或症状轻微。
2. 急性加重期：以呼吸道症状急性恶化（呼吸困难和/或咳嗽、咳痰加重）为特征的事件，通常持续14天以内，多与呼吸道感染、空气污染等有关。

并发症：
包括慢性呼吸衰竭、自发性气胸、慢性肺源性心脏病等。

四、诊断与鉴别诊断

诊断标准：
COPD的诊断需综合危险因素暴露史、症状、体征及肺功能检查等临床资料，并排除可引起类似症状和持续气流受限的其他疾病。肺功能检查表现为持续气流受限是确诊COPD的必备条件。

核心诊断标准：吸入支气管舒张剂后，第一秒用力呼气容积（FEV1）与用力肺活量（FVC）的比值（FEV1/FVC）< 0.7，即可明确存在持续的气流受限。这是诊断COPD的“金标准”。FEV1占预计值的百分比可用于评价气流受限的严重程度。

辅助检查：
1. 肺功能检查：除诊断外，还用于疾病严重程度分级、预后及治疗反应评估。对于高危人群，建议每年进行一次肺通气功能检查。
2. 胸部X线检查：可显示肺过度充气、肺野透亮度增高等征象，并协助发现肺部并发症。
3. 胸部CT检查：高分辨率CT可清晰显示肺气肿类型及程度，有助于鉴别诊断。
4. 脉搏血氧饱和度监测和动脉血气分析：用于评估是否存在低氧血症、高碳酸血症及呼吸衰竭。
5. 血常规：稳定期外周血嗜酸性粒细胞（EOS）计数对治疗方案选择（如是否加用吸入性糖皮质激素）有参考意义。

鉴别诊断要点：
主要需与支气管哮喘、充血性心力衰竭、支气管扩张症、肺结核等疾病相鉴别。其中，与哮喘的鉴别尤为重要。哮喘的气流受限多为可逆性（经治疗可恢复正常），而COPD的气流受限则为持续性且进行性发展。部分长期未规律治疗的哮喘患者可出现不可逆气流受限，此时鉴别困难，需结合病史、症状变异性及对治疗的反应等综合判断。

五、治疗方案

治疗原则：
COPD的治疗目标是减轻当前症状、提高运动耐力和改善健康状况，以及降低未来风险（预防疾病进展、预防和治疗急性加重、降低病死率）。治疗应遵循个体化原则。

药物治疗：
1. 支气管舒张剂：是COPD治疗的核心药物。按需或规律使用可预防或减轻症状。
    - β2受体激动剂：短效（SABA）如沙丁胺醇用于急性缓解；长效（LABA）如沙美特罗、福莫特罗用于长期维持治疗。
    - 抗胆碱能药物：短效（SAMA）如异丙托溴铵；长效（LAMA）如噻托溴铵，是COPD稳定期维持治疗的基石药物之一。
    - 茶碱类药物：需监测血药浓度，注意其不良反应。
2. 糖皮质激素：
    - 吸入性糖皮质激素（ICS）：不推荐单独使用，通常与LABA联合（如布地奈德/福莫卡松）用于特定表型（如血EOS升高）的中重度COPD患者，可减少急性加重。长期使用需注意口腔念珠菌感染、肺炎风险等。
    - 全身性激素：口服泼尼松或静脉使用甲泼尼龙，主要用于COPD急性加重期的短期治疗。
3. 磷酸二酯酶-4（PDE4）抑制剂：如罗氟司特，适用于伴有慢性支气管炎表型和重度气流受限的患者，可降低急性加重风险。
4. 祛痰药：如乙酰半胱氨酸、羧甲司坦、氨溴索，用于痰液粘稠不易咳出者。
5. 抗生素：仅限用于治疗感染性急性加重或具有细菌感染征象时，不推荐长期预防性使用。

最新治疗进展（2024-2025年）：
近年来，COPD治疗领域迎来了突破性进展，多款具有新作用机制的药物获批上市。
1. 恩司芬群（Ensifentrine）：于2024年在美国获批，是20多年来首个用于COPD维持治疗的新型机制吸入产品。作为PDE3和PDE4选择性双重抑制剂，它兼具支气管扩张和非甾体抗炎作用，可改善症状并减少急性加重，推动治疗进入“去激素化”时代。2025版GOLD指南已推荐其用于使用二联长效支气管舒张剂后仍有呼吸困难的患者。
2. 度普利尤单抗（Dupilumab）：一种抗IL-4/IL-13单抗，于2024年在中国获批，用于以血嗜酸性粒细胞升高为特征的不受控制的COPD患者的附加维持治疗。它是全球首款获批COPD适应症的靶向生物制剂。
3. 美泊利珠单抗（Mepolizumab）：一种抗IL-5单抗，于2025年在中国获批，用于嗜酸性粒细胞表型COPD的附加维持治疗，是COPD领域首个可每月给药一次的生物制剂。
此外，针对TSLP、IL-33等靶点的生物制剂以及国内企业研发的PDE3/4抑制剂（如正大天晴的TQC3721）也处于临床研发阶段。

非药物治疗：
非药物治疗是COPD管理的重要组成部分，可改善生活质量、减轻症状。
1. 戒烟：是所有吸烟COPD患者最重要且最有效的干预措施，能显著减缓肺功能下降。
2. 肺康复：包括运动训练（如步行、骑自行车）、呼吸训练（如缩唇呼吸、腹式呼吸）和健康教育，可改善运动耐力和呼吸困难。
3. 长期家庭氧疗：适用于伴有严重静息性低氧血症（PaO2 ≤ 55 mmHg）的患者，每天吸氧时间>15小时，可提高生存率。
4. 营养支持：维持健康体重，营养不良者需加强营养。
5. 心理干预：针对伴发的焦虑、抑郁情绪提供支持。

六、预后与康复

预后评估：
COPD的预后与气流受限的严重程度（FEV1值）、急性加重的频率和严重程度、合并症（如心血管疾病、肺癌、骨质疏松、焦虑抑郁）、营养状况及体重指数（BMI）等多种因素相关。需对患者进行综合评估，包括症状评估（如采用改良版英国医学研究委员会呼吸问卷mMRC或慢阻肺患者自我评估测试CAT）、肺功能分级和急性加重风险评估，以指导个体化治疗。

康复指导：
1. 运动指导：制定个性化的有氧运动和抗阻训练计划，循序渐进。
2. 呼吸锻炼：指导患者坚持进行缩唇呼吸和腹式呼吸训练，改善通气效率。
3. 日常生活管理：节省体能技术、营养膳食指导、情绪管理。
4. 自我管理教育：帮助患者识别急性加重的早期征象并知晓应对措施。

随访管理：
COPD需要长期、规律的随访。随访内容应包括：
1. 症状变化及治疗依从性评估。
2. 肺功能监测：定期复查肺功能以评估疾病进展。
3. 药物吸入技术评估与纠正。
4. 急性加重事件记录与原因分析。
5. 合并症筛查与管理。
6. 疫苗接种情况（流感疫苗、肺炎球菌疫苗）确认。

七、预防策略

一级预防（病因预防）：
1. 控制吸烟：首要措施。加强控烟宣传，创建无烟环境，提供戒烟服务。
2. 改善环境：加强职业防护，减少有害气体和颗粒暴露；改善室内外空气质量。
3. 防治呼吸道感染：重视婴幼儿和儿童期的下呼吸道感染防治。
4. 加强锻炼：增强体质，提高免疫力。

二级预防（“三早”预防）：
1. 早期筛查：对高危人群（如长期吸烟者、职业暴露者、有COPD家族史者、反复呼吸道感染者）定期进行肺功能检查。可采用筛查问卷（如COPD-SQ）进行初步识别。
2. 早期诊断：提高医务人员和公众对COPD的认知，对有慢性咳嗽、咳痰、呼吸困难症状者及早进行肺功能检查。
3. 早期干预：对确诊的早期患者，立即启动戒烟、避免危险因素暴露等干预，并根据病情开始规范药物治疗。

高危人群管理：
建立高危人群健康档案，进行定期随访和健康教育，督促其改变不良生活方式，坚持肺功能监测，一旦出现症状或肺功能下降，及时转诊至专科进行诊治。
