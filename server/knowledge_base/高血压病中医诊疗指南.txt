高血压病中医诊疗指南（2019年版）

一、疾病概述
高血压病在中医属"眩晕"、"头痛"、"肝风"等范畴。主要病机为肝阳上亢、阴虚阳亢、痰湿中阻、瘀血阻络等。中医治疗强调辨证论治，整体调节。

二、病因病机
1. 肝阳上亢：情志不遂，肝失疏泄，肝阳上扰清窍
2. 阴虚阳亢：肾阴亏虚，水不涵木，肝阳上亢
3. 痰湿中阻：饮食不节，脾失健运，痰湿内生，上蒙清窍
4. 瘀血阻络：久病入络，气血瘀滞，脑脉不通
5. 气血两虚：劳倦过度，气血亏虚，清窍失养
6. 肾精不足：年老体衰，肾精亏虚，髓海不足

三、辨证分型与治疗
1. 肝阳上亢证
   - 主症：头晕胀痛，面红目赤，急躁易怒
   - 治法：平肝潜阳，清火熄风
   - 方药：天麻钩藤饮加减
   - 常用药：天麻、钩藤、石决明、栀子、黄芩、牛膝、杜仲、益母草、桑寄生、夜交藤、茯神

2. 阴虚阳亢证
   - 主症：头晕耳鸣，腰膝酸软，五心烦热
   - 治法：滋阴潜阳，平肝熄风
   - 方药：镇肝熄风汤加减
   - 常用药：怀牛膝、生赭石、生龙骨、生牡蛎、生龟板、生杭芍、玄参、天冬、川楝子、生麦芽、茵陈、甘草

3. 痰湿中阻证
   - 主症：头晕头重，胸闷恶心，食少多寐
   - 治法：化痰祛湿，健脾和胃
   - 方药：半夏白术天麻汤加减
   - 常用药：半夏、白术、天麻、茯苓、橘红、甘草、生姜、大枣

4. 瘀血阻络证
   - 主症：头晕头痛，痛有定处，舌质紫暗
   - 治法：活血化瘀，通络止痛
   - 方药：血府逐瘀汤加减
   - 常用药：桃仁、红花、当归、生地、川芎、赤芍、牛膝、桔梗、柴胡、枳壳、甘草

5. 气血两虚证
   - 主症：头晕目眩，面色苍白，心悸失眠
   - 治法：补益气血，调养心脾
   - 方药：归脾汤加减
   - 常用药：人参、白术、茯苓、甘草、当归、黄芪、酸枣仁、远志、龙眼肉、木香

6. 肾精不足证
   - 主症：头晕耳鸣，腰膝酸软，精神萎靡
   - 治法：补肾填精，滋养髓海
   - 方药：左归丸加减
   - 常用药：熟地、山药、山茱萸、枸杞子、菟丝子、鹿角胶、龟板胶、川牛膝

四、针灸治疗
1. 主穴：风池、太冲、曲池、足三里、三阴交
2. 配穴：
   - 肝阳上亢：加行间、侠溪
   - 阴虚阳亢：加太溪、肾俞
   - 痰湿中阻：加丰隆、中脘
   - 瘀血阻络：加血海、膈俞
3. 治疗方法：平补平泻，留针20-30分钟，每周2-3次

五、其他疗法
1. 耳穴压豆：取穴降压沟、神门、心、肝、肾、交感
2. 推拿按摩：揉按太阳穴、百会穴，推桥弓穴
3. 中药足浴：使用钩藤、夏枯草、桑叶、菊花等煎汤泡脚
4. 食疗药膳：
   - 菊花茶：菊花10g，代茶饮
   - 决明子粥：决明子15g，粳米50g，煮粥
   - 山楂荷叶茶：山楂15g，荷叶10g，代茶饮

六、生活方式干预
1. 情志调摄：保持心情舒畅，避免情绪激动
2. 饮食调理：低盐低脂，多吃蔬菜水果，忌食肥甘厚味
3. 适量运动：太极拳、八段锦、散步等柔和运动
4. 起居有常：保证充足睡眠，避免熬夜
5. 戒烟限酒：戒烟，限制饮酒

七、预防与调护
1. 定期监测血压，记录血压日记
2. 遵医嘱规律服药，不可随意停药
3. 注意气候变化，避免受寒感冒
4. 预防便秘，保持大便通畅
5. 注意药物不良反应，定期复诊
