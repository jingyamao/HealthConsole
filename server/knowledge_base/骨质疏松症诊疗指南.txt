骨质疏松症诊疗指南

一、概述

骨质疏松症是一种以骨量低下、骨组织微结构损坏，导致骨脆性增加，易发生骨折为特征的全身性骨病。根据病因，可分为原发性骨质疏松症（包括绝经后骨质疏松症、老年性骨质疏松症和特发性骨质疏松症）和继发性骨质疏松症。

流行病学数据显示，骨质疏松症是中老年群体的常见疾病。2018年我国65岁及以上人群骨质疏松症患病率达到32%，患病总人数约为9千万，其中女性约7千万，显著多于男性。随着人口老龄化加剧，其患病率逐年升高，骨质疏松性骨折已成为老年人群致残和致死的主要原因之一，给社会和家庭带来沉重的疾病负担与经济压力。

二、病因和发病机制

骨质疏松症的主要病因涉及多种因素。其核心发病机制是骨重建失衡，即破骨细胞介导的骨吸收与成骨细胞介导的骨形成之间的动态平衡被破坏，导致骨吸收大于骨形成，骨量逐渐丢失，骨微结构损坏。

主要的危险因素包括：
1.  不可控因素：年龄增长、女性绝经后雌激素水平下降、种族（白种人及黄种人风险较高）、脆性骨折家族史等。
2.  可控因素：钙和/或维生素D摄入不足、长期缺乏运动或卧床、吸烟、过量饮酒、饮用过多咖啡及碳酸饮料等。
3.  疾病与药物因素：内分泌疾病（如甲状腺功能亢进症、性腺功能减退）、消化系统疾病（如吸收不良综合征）、慢性肾脏病、长期使用糖皮质激素、抗癫痫药等。

三、临床表现

骨质疏松症早期常无明显症状，因此被称为“寂静的疾病”。随着病情进展，可能出现以下表现：

1.  症状和体征：
    *   疼痛：以腰背痛或周身骨痛多见，负荷增加时疼痛加重或活动受限。
    *   脊柱变形：严重者可出现身高缩短、驼背、脊柱后凸或侧弯畸形。
    *   骨折：即脆性骨折，指在日常生活中受到轻微外力（如跌倒、甚至弯腰、咳嗽）即可发生的骨折。常见部位为椎体（胸、腰椎）、髋部（股骨近端）和前臂远端。

2.  临床分型：主要依据病因分为原发性和继发性两大类。原发性又细分为I型（绝经后骨质疏松，主要由于雌激素缺乏）、II型（老年性骨质疏松，与年龄增长相关的退化）和特发性骨质疏松（多见于青壮年或青少年）。

3.  并发症：最严重的并发症是骨质疏松性骨折，尤其是髋部骨折，被称为“人生最后一次骨折”，其致残率高，严重影响患者生活质量，并可能导致死亡风险增加。

四、诊断与鉴别诊断

诊断需结合临床表现、骨密度测量、影像学检查和骨折风险评估进行综合判断。

1.  诊断标准（引用权威指南）：
    *   骨密度测量：双能X线吸收测定法（DXA）是诊断骨质疏松症的金标准。通常测量腰椎1-4和股骨颈的骨密度。诊断标准基于世界卫生组织（WHO）推荐的T值（与同性别、同种族健康年轻人的骨峰值比较）：
        *   T值 ≥ -1.0：骨量正常
        *   -2.5 < T值 < -1.0：骨量减少（或称低骨量）
        *   T值 ≤ -2.5：骨质疏松症
    *   脆性骨折史：对于绝经后女性或50岁及以上男性，一旦发生脆性骨折，即使骨密度未达上述标准，也可临床诊断为骨质疏松症。

2.  辅助检查：
    *   影像学检查：X线平片可观察骨折和脊柱变形，但骨量丢失超过30%时才能在X线上显现。定量CT（QCT）能更精确地测量体积骨密度，尤其适用于评估椎体松质骨。
    *   实验室检查：包括血钙、磷、碱性磷酸酶等基本骨代谢指标；骨转换标志物（如血清Ⅰ型原胶原N-端前肽PINP和血清Ⅰ型胶原C-末端肽交联CTX）有助于判断骨转换类型、评估骨折风险及监测药物疗效。此外，检查25-羟维生素D、甲状旁腺激素（PTH）、性激素、甲状腺功能等，有助于鉴别继发性骨质疏松症。
    *   骨折风险评估：采用FRAX等工具，综合临床危险因素（年龄、性别、体重指数、既往骨折史、父母髋部骨折史、吸烟、饮酒、糖皮质激素使用、类风湿关节炎等）和股骨颈骨密度，计算未来10年主要骨质疏松性骨折及髋部骨折的概率。

3.  鉴别诊断要点：需与其他可导致低骨量或骨折的疾病相鉴别，包括：
    *   内分泌疾病：如甲状旁腺功能亢进症、库欣综合征、性腺功能减退症、甲状腺功能亢进症等。
    *   风湿性疾病：如类风湿关节炎。
    *   消化系统疾病：如胃大部切除术后、炎症性肠病、乳糜泻等导致的吸收不良。
    *   慢性肾脏疾病。
    *   血液系统疾病：如多发性骨髓瘤。
    *   药物性因素：长期使用糖皮质激素、肝素、抗癫痫药等。
    *   其他：成骨不全症、变形性骨炎（Paget病）等。

五、治疗方案

治疗目标是缓解骨痛、增加骨密度、改善骨质量、降低骨折风险、提高生活质量。

1.  治疗原则：
    *   基础措施：贯穿始终。
    *   药物干预：根据骨折风险分层选择药物。对于高风险和极高风险患者，应优先选择强效抗骨质疏松药物。
    *   康复治疗：改善功能，预防跌倒。
    *   长期管理：注重用药依从性、安全性和定期随访。

2.  药物治疗：
    *   基础治疗药物：
        *   钙剂：推荐每日元素钙摄入量为800-1200mg，膳食摄入不足部分需补充。常用制剂有碳酸钙、枸橼酸钙等。
        *   维生素D：推荐普通维生素D（如维生素D3）每日摄入量为800-1200 IU。对于维生素D缺乏者，需先纠正缺乏。活性维生素D及其类似物（如骨化三醇、阿法骨化醇）可用于肝肾功能减退患者，或与普通维生素D联用。

    *   抗骨吸收药物（抑制破骨细胞活性）：
        *   双膦酸盐类：是目前应用最广泛的一线药物。代表药物：
            *   阿仑膦酸钠：口服，每周一次70mg或每日一次10mg。
            *   唑来膦酸：静脉滴注，每年一次5mg。
            *   利塞膦酸钠：口服，每周一次35mg或每日一次5mg。
            *   注意事项：口服制剂可能引起胃肠道不适，需空腹、大量清水送服并保持直立至少30分钟。静脉制剂需注意流感样症状和肾功能监测。长期使用（通常超过3-5年）需关注非典型股骨骨折和下颌骨坏死风险，需定期评估。
        *   RANKL抑制剂：
            *   地舒单抗：皮下注射，每6个月一次60mg。不经肾脏代谢，适用于肾功能不全患者。需长期规律用药，停药后骨密度可能快速下降，骨折风险反弹。
        *   选择性雌激素受体调节剂（SERMs）：
            *   雷洛昔芬：口服，每日一次60mg。适用于绝经后女性，尤其伴有乳腺癌风险考量者。可能增加静脉血栓栓塞风险。
        *   降钙素类：
            *   鲑降钙素：鼻喷或注射剂型。能缓解骨质疏松性骨痛，但因其长期使用可能增加肿瘤风险，一般不作为长期治疗首选，可用于短期镇痛。

    *   促骨形成药物（刺激成骨细胞活性）：
        *   甲状旁腺素类似物：
            *   特立帕肽：皮下注射，每日一次20μg。适用于高骨折风险患者，疗程一般不超过24个月。治疗结束后需序贯使用抗骨吸收药物以维持骨密度。
        *   硬骨素抑制剂（新型双重作用药物）：
            *   罗莫索珠单抗：2025年12月获中国国家药监局批准，用于治疗骨折高危的绝经后女性骨质疏松症。其通过抑制硬骨素，同时促进骨形成和抑制骨吸收。用法为每月一次皮下注射210mg，疗程12个月。临床研究显示其能显著降低椎体骨折风险并增加骨密度。但需特别注意，该药有增加主要不良心血管事件（心肌梗死、卒中、心血管死亡）风险的黑框警告，禁用于有心肌梗死或卒中病史的患者。

3.  非药物治疗：
    *   运动疗法：推荐进行规律的负重运动（如快走、慢跑、跳舞）和抗阻训练（如举重、弹力带），每周至少3-5次，每次30分钟，以增强肌肉力量和骨强度。平衡训练（如太极拳）有助于降低跌倒风险。
    *   营养支持：保证充足的蛋白质摄入，均衡膳食，多摄入富含钙和维生素D的食物（如牛奶、酸奶、豆制品、深绿色蔬菜、海产品）。
    *   预防跌倒：是减少骨折的关键环节。措施包括评估居家环境安全（如防滑、增加照明、安装扶手）、治疗影响视力和平衡的疾病、穿防滑鞋、必要时使用髋部保护器等。
    *   康复治疗：对于已发生骨折或功能障碍者，应在康复医师指导下进行物理治疗、作业治疗、使用康复辅具等，以改善疼痛、恢复功能、提高生活质量。

4.  最新治疗进展：
    *   治疗策略更强调根据骨折风险分层进行个体化选择。
    *   序贯和联合治疗：对于极高危患者，可采用促骨形成药物（如特立帕肽或罗莫索珠单抗）治疗后，序贯抗骨吸收药物（如地舒单抗或双膦酸盐）的长期管理方案。
    *   数字健康管理：整合移动应用程序进行用药提醒、骨健康教育、跌倒风险评估和生活方式干预，形成药物治疗与数字疗法结合的综合管理方案。
    *   研究方向：包括开发口服生物制剂、靶向递送系统、Wnt信号通路调节剂等新靶点药物，以及利用AI模型进行动态风险评估和治疗方案调整。

六、预后与康复

1.  预后评估：预后与是否发生骨折、骨折部位、是否得到及时规范的治疗及康复密切相关。髋部骨折后1年内，死亡率可达20%，约50%患者致残，生活无法自理。坚持规范治疗可有效降低再骨折风险。
2.  康复指导：应遵循个体化、循序渐进的原则。包括：
    *   疼痛管理：采用物理因子治疗（如脉冲电磁场）、必要时在医生指导下使用镇痛药。
    *   运动康复：在专业指导下进行，重点加强核心肌群和下肢力量训练、平衡训练。
    *   功能训练：针对日常生活活动能力进行训练。
    *   心理支持：关注患者的焦虑、抑郁情绪，必要时提供心理干预。
3.  随访管理：
    *   治疗依从性监测：定期评估患者用药情况，解决用药障碍。
    *   疗效监测：每1-2年复查骨密度（DXA）。骨转换标志物可在治疗后3-6个月复查，以早期评估药物反应。
    *   安全性监测：定期评估药物不良反应（如下颌骨症状、大腿疼痛等）。对于使用双膦酸盐或地舒单抗长期治疗的患者，需定期进行风险获益再评估。
    *   骨折风险再评估：定期使用FRAX工具更新骨折风险。

七、预防策略

遵循“三级预防”原则。

1.  一级预防（无病防病）：
    *   目标：提高峰值骨量，减缓骨量丢失，预防骨质疏松症和首次骨折的发生。
    *   措施：
        *   全人群健康教育：普及骨健康知识。
        *   从儿童青少年期开始：保证充足钙和维生素D摄入，积极参加体育锻炼，以获得理想的峰值骨量。
        *   成年及中年期：坚持健康生活方式，均衡营养，规律运动，戒烟限酒，保持适宜体重。
        *   围绝经期及老年期：在医生指导下，必要时进行绝经激素治疗或抗骨质疏松药物干预，以预防骨量快速丢失。

2.  二级预防（有病早治）：
    *   目标：早期发现和治疗骨量减少及骨质疏松症患者，防止病情进展和骨折发生。
    *   措施：
        *   高危人群筛查：对65岁以上女性、70岁以上男性、有脆性骨折史的成年人、长期使用糖皮质激素者等，定期进行骨密度检测。
        *   早期诊断与干预：对确诊的骨量减少和骨质疏松症患者，及时启动包括基础措施和药物治疗在内的综合管理。

3.  高危人群管理：
    *   明确高危人群：包括绝经后女性、老年男性、有骨折家族史者、低体重者、有不良生活习惯者、患有相关疾病或服用相关药物者等。
    *   强化管理：对此类人群进行重点健康教育，鼓励其定期筛查，积极控制危险因素，一旦确诊，应进行更积极的治疗和更密切的随访。
