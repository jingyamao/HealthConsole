一、概述

血脂异常，俗称高脂血症，是指血浆中总胆固醇（TC）、甘油三酯（TG）、低密度脂蛋白胆固醇（LDL-C）水平升高和/或高密度脂蛋白胆固醇（HDL-C）水平降低的一种血脂代谢紊乱状态。它是动脉粥样硬化性心血管疾病（ASCVD）最重要且可干预的危险因素之一。

流行病学数据显示，近三十年来，我国人群的血脂水平呈逐步升高趋势，血脂异常患病率明显增加。血脂异常与动脉粥样硬化、冠心病、脑卒中等心脑血管疾病的发生发展密切相关。有效控制血脂异常，对于降低我国ASCVD的发病及死亡风险、减轻疾病负担具有至关重要的公共卫生意义。然而，目前我国成人血脂异常患者的知晓率、治疗率和控制率仍处于较低水平，防治工作亟待加强。

二、病因和发病机制

血脂异常的病因可分为原发性和继发性两大类。

1. 原发性血脂异常：主要由遗传因素决定，通常具有家族聚集性，例如家族性高胆固醇血症（FH）等。
2. 继发性血脂异常：更为常见，主要由后天环境因素和疾病状态引起。主要包括：
    * 不良生活方式：如高脂肪、高胆固醇、高糖饮食，缺乏体力活动，吸烟，过量饮酒等。
    * 疾病因素：如糖尿病、甲状腺功能减退症、肥胖症、肾病综合征、肝脏疾病等。
    * 药物影响：如某些利尿剂、β受体阻滞剂、糖皮质激素等。

其病理生理机制核心在于脂蛋白代谢过程的异常，包括合成、转运和清除环节的失衡，导致富含胆固醇和/或甘油三酯的脂蛋白在血液中蓄积。

三、临床表现

血脂异常本身通常没有明显的症状和体征，多数患者是在常规血液生化检查中发现的。其主要的临床意义在于长期血脂异常所导致的靶器官损害和并发症。

1. 临床分型：根据升高的血脂成分不同，主要分为四种类型：
    * 高胆固醇血症：以血清总胆固醇（TC）和/或低密度脂蛋白胆固醇（LDL-C）水平升高为主。
    * 高甘油三酯血症：以血清甘油三酯（TG）水平升高为主。
    * 混合型高脂血症：血清TC和TG水平均升高。
    * 低高密度脂蛋白胆固醇血症：血清高密度脂蛋白胆固醇（HDL-C）水平降低。

2. 并发症：长期未加控制的血脂异常，特别是LDL-C升高，是导致动脉粥样硬化的主要原因。其并发症主要表现为各种ASCVD，包括：
    * 冠状动脉粥样硬化性心脏病（冠心病），可导致心绞痛、心肌梗死。
    * 缺血性脑卒中。
    * 外周动脉疾病。
    * 此外，严重的高甘油三酯血症（通常TG > 11.3 mmol/L）有诱发急性胰腺炎的风险。

四、诊断与鉴别诊断

（一）诊断标准

血脂异常的诊断主要基于空腹（禁食12-14小时）血脂检测结果。根据《中国成人血脂异常防治指南（2016年修订版）》及国际共识，诊断切点如下：

1. 总胆固醇（TC）：
    * 合适范围：< 5.2 mmol/L (200 mg/dL)
    * 边缘升高：≥ 5.2 mmol/L 且 < 6.2 mmol/L (200-239 mg/dL)
    * 升高：≥ 6.2 mmol/L (240 mg/dL)

2. 低密度脂蛋白胆固醇（LDL-C）：“坏胆固醇”，是首要干预靶点。
    * 合适范围：< 3.4 mmol/L (130 mg/dL)
    * 边缘升高：≥ 3.4 mmol/L 且 < 4.1 mmol/L (130-159 mg/dL)
    * 升高：≥ 4.1 mmol/L (160 mg/dL)

3. 高密度脂蛋白胆固醇（HDL-C）：“好胆固醇”。
    * 合适范围：≥ 1.0 mmol/L (40 mg/dL)
    * 升高：≥ 1.6 mmol/L (60 mg/dL)
    * 减低：< 1.0 mmol/L (40 mg/dL)

4. 甘油三酯（TG）：
    * 合适范围：< 1.7 mmol/L (150 mg/dL)
    * 边缘升高：≥ 1.7 mmol/L 且 < 2.3 mmol/L (150-199 mg/dL)
    * 升高：≥ 2.3 mmol/L (200 mg/dL)

（二）诊断更新要点（基于2026 ACC/AHA指南）

1. 风险评估更新：推荐使用更新的PREVENT-ASCVD风险评估方程，用于估算30-79岁成人10年及30年ASCVD风险，以指导治疗决策。
2. 常规检测建议：强烈推荐所有成年人终身至少检测一次脂蛋白(a) [Lp(a)]，以识别ASCVD高风险个体。对于甘油三酯升高、糖尿病或LDL-C已达标但存在残余风险的患者，检测载脂蛋白B（ApoB）有助于改善风险评估。
3. 冠状动脉钙化（CAC）评分：对于中等风险人群，CAC评分成为指导是否启动降脂治疗的Ⅰ类推荐工具。

（三）鉴别诊断

重点在于鉴别原发性与继发性血脂异常。需详细询问病史、用药史，并进行相关检查以排除糖尿病、甲状腺功能减退、肾病综合征、肝胆疾病等常见继发原因。

五、治疗方案

（一）治疗原则

血脂异常的治疗是降低ASCVD风险的关键。治疗策略需根据个体的心血管风险分层来制定，首要目标是降低LDL-C水平。治疗性生活方式改变（TLC）是所有患者治疗的基础，贯穿始终。对于心血管高风险及极高风险人群，在生活方式干预基础上，需尽早启动药物治疗，并设定更严格的血脂控制目标。

（二）非药物治疗（治疗性生活方式改变）

这是血脂异常管理的基石和一线措施，无论是否用药都必须长期坚持。

1. 饮食治疗：原则是“四低一高”，即低热量、低脂肪、低胆固醇、低糖、高纤维膳食。
    * 限制饱和脂肪（如动物脂肪、棕榈油）和反式脂肪（如加工零食、油炸食品）摄入。
    * 胆固醇摄入每日应少于300毫克。
    * 增加膳食纤维摄入，特别是可溶性膳食纤维，如燕麦、豆类、水果和蔬菜。
    * 选用富含不饱和脂肪酸的食物，如鱼类、坚果、橄榄油等。推荐地中海饮食模式。
    * 食用油以植物油为主，每人每天用量不超过25-30克。

2. 规律运动：建议每周进行至少150分钟的中等强度有氧运动（如快走、游泳、骑自行车）或75分钟的高强度有氧运动。运动应循序渐进、持之以恒。

3. 控制体重：肥胖，尤其是中心型肥胖，与血脂异常密切相关。超重或肥胖者减轻体重（目标为体重下降5%-10%）可显著改善血脂谱。

4. 戒烟限酒：吸烟会升高TG、降低HDL-C，戒烟1年可使HDL-C恢复至不吸烟者水平，冠心病风险降低50%。应严格限制饮酒。

（三）药物治疗

当生活方式干预不能使血脂达标时，需启动药物治疗。

1. 治疗目标：根据ASCVD风险分层设定个体化目标。
    * 低危人群：LDL-C < 3.4 mmol/L。
    * 中、高危人群：LDL-C < 2.6 mmol/L (或 < 100 mg/dL)。
    * 极高危人群（如已确诊ASCVD）：LDL-C < 1.8 mmol/L (或 < 70 mg/dL)，部分极高危者需降至 < 1.4 mmol/L (或 < 55 mg/dL)。

2. 主要降脂药物：
    * 他汀类药物：是降胆固醇治疗的基石和首选药物。通过抑制胆固醇合成降低LDL-C。常用药物有阿托伐他汀、瑞舒伐他汀等。起始治疗可根据风险选择中等强度或高强度他汀。
    * 胆固醇吸收抑制剂：如依折麦布，常与他汀联用，用于单用他汀LDL-C不达标或不能耐受大剂量他汀的患者。
    * 前蛋白转化酶枯草溶菌素9（PCSK9）抑制剂：如依洛尤单抗、阿利西尤单抗，降LDL-C效力强，适用于他汀联合依折麦布治疗后仍不达标的极高危患者，或家族性高胆固醇血症患者。
    * 贝特类药物：主要降低TG，适用于高甘油三酯血症患者。常用非诺贝特。
    * 高纯度ω-3脂肪酸制剂：（如二十碳五烯酸乙酯）可用于降低TG。
    * 其他：包括胆酸螯合剂、烟酸类、普罗布考等，但临床应用相对较少。

3. 联合用药：对于单药治疗不达标或混合型血脂异常患者，常需联合用药。常见方案包括：他汀 + 依折麦布；他汀 + PCSK9抑制剂；他汀 + 贝特类（需警惕肌病和肝损伤风险）等。2026年指南更强调根据风险分层早期、积极地启动联合治疗。

（四）其他治疗与最新进展

1. 脂蛋白血浆置换：适用于药物治疗无效的纯合子型家族性高胆固醇血症等极少数情况。
2. 新型药物：包括小干扰RNA（siRNA）药物英克司兰（Inclisiran），每年仅需皮下注射两次，用于降低LDL-C；针对脂蛋白(a) [Lp(a)] 升高的新药正在研发中。

六、预后与康复

1. 预后评估：血脂异常患者的预后主要取决于血脂水平、其他心血管危险因素（如高血压、糖尿病、吸烟）的控制情况以及是否发生ASCVD事件。将LDL-C控制在目标范围内，可显著降低心肌梗死、脑卒中等事件的风险和死亡率。
2. 康复指导：对于已发生ASCVD事件的患者，除严格二级预防用药外，应在医生指导下进行循序渐进的心脏康复训练，包括运动指导、生活方式重塑和心理支持。
3. 随访管理：血脂异常需长期甚至终身管理。建议每3-6个月复查一次血脂、肝功能、肌酸激酶等指标，评估疗效和药物安全性。同时定期监测血压、血糖等其他危险因素。

七、预防策略

1. 一级预防：针对尚未发生ASCVD的全体人群。
    * 健康生活方式推广：倡导均衡饮食、规律运动、保持健康体重、戒烟限酒。
    * 定期筛查：建议20-40岁成年人至少每5年检测一次血脂；40岁以上男性和绝经后女性应每年检测。有早发心血管病家族史者应提前并加强筛查。
    * 风险评估与分层管理：利用风险评估工具识别中高危人群，及早进行生活方式强化干预，必要时启动药物治疗。

2. 二级预防：针对已确诊ASCVD（如冠心病、缺血性卒中）的患者。
    * 目标是防止疾病复发、降低死亡率。
    * 必须在生活方式干预基础上，使用他汀等药物进行强化降脂治疗，达到更严格的LDL-C目标值（如<1.8 mmol/L或<1.4 mmol/L）。
    * 严格控制其他危险因素，并坚持长期、规律的随访和治疗。

3. 高危人群管理：
    * 包括糖尿病患者、慢性肾脏病患者、高血压患者、肥胖者、有早发心血管病家族史者等。
    * 这些人群的血脂控制目标通常比普通人更严格。
    * 需进行更频繁的监测和更积极的干预。
